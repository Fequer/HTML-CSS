Principal Formatting

Bold / Highlight(MARK)

In this phrase, we have a term in Bold using the tag <B> (not semantic)

In this phrase, we have a term in Highlighted using the tag <STRONG> (semantic)

Italics / Emphasis

In this phrase, we have a term in Italics using the tag <I> (not semantic)

In this phrase, we have a term in Emphasis using the tag <EM> (semantic)

Marked Text

We have build also a marked text using the tag <MARK>

And in another paragraph, we have another mark text at the end

Big and small Text

We are build a big text and a small text that paragraph.

Text deleted

We can mark a text as deleted to indicate that it has been read, but not considered

Text inserted

We can mark a text as inserted to give an emphasis and indicate that it was added later.

Exists also the underline with a tag U (not semantic)

Text Subscript
